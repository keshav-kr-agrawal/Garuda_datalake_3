# Changelog

## [1.0.0] - 2026-06-05

### Added
- `FaceEmbedder` — platform-agnostic face embedding and 1:N verification engine
- `LivenessDetector` — EAR/MAR liveness math, pose estimation, challenge-response anti-spoofing
- `EnrollmentEngine` — multi-angle enrollment state machine
- `CLAHEProcessor` — CLAHE image contrast preprocessing (Canvas-based, no native deps)
- `AuditLedger` — pure-JS SHA-256 cryptographic blockchain audit trail
- `LocalDatabase` — vector search and multi-angle biometric lookup
- `SqliteWebAdapter` — SQLite WASM storage adapter (browser)
- `AsyncStorageAdapter` — AsyncStorage adapter (React Native)
- `DatalakeApiClient` — NIC Datalake 3.0 API bridge
- `AwsAuthClient` — AWS Cognito authentication client
- `AwsSyncClient` — AWS DynamoDB sync client
- Full TypeScript type definitions
